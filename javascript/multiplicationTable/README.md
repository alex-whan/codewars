## Task

Create an `NxN` multiplication table, of size provided in parameter `size`.

**Example:**

For example, when given `size` is `3`, the table should be conceptualized as follows:

    1 2 3
    2 4 6
    3 6 9

For this example, the return value should be:

    [[1,2,3],[2,4,6],[3,6,9]]

OR:

    [[1,2,3],
    [2,4,6],
    [3,6,9]]
